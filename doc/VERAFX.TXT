=== VERAFX - VERA FX accelerator ===

VERA FX (r47+ firmware) adds hardware helpers: a 16x16 multiplier, a 32-bit
write cache (fast VRAM fills), transparent writes, and line/polygon/affine
address helpers.

Built-in native words:
  DCSEL   ( n -- )          select DCSEL bank 0-63 so the FX registers at
                            $9F29-$9F2C can be reached with C! / C@.
  FX-MULT ( a b -- lo hi )  signed 16x16 -> 32-bit multiply (result = a double).
      1000 1000 FX-MULT SWAP . .  ->  16960 15  (= 1000000)

For more, load the toolkit (HELP requires ASSEMBLER.FTH first):
  S" ASSEMBLER.FTH" INCLUDED   S" VERAFX.FTH" INCLUDED
  FX*      ( n1 n2 -- d )                signed multiply -> double
  FX-FILL  ( byte vbank vaddr count -- ) fast 32-bit-cache VRAM fill (~4x)
  FX-CLEAR ( vbank vaddr count -- )      zero a VRAM region
  FX-DCSEL ( n -- ) / FX-OFF ( -- )      low-level plumbing for line/poly/affine

See also the ASMGFX.FTH graphics toolkit, which uses VERA FX for its fills.
